# Payment Systems Guide

## Recommended Providers

### For EU (France)
| Provider | Type | Fees | Notes |
|----------|------|------|-------|
| LemonSqueezy | Payments | 5% + 50c | Best for SaaS |
| Stripe | Payments | 1.4% + 25c | Industry standard |
| PayPal | Payments | 2.9% + 35c | Consumer trust |

### For Crypto
| Provider | Type | Fees | Notes |
|----------|------|------|-------|
| Coinbase Commerce | Crypto | 0% | Easy setup |
| NOWPayments | Crypto | 0.4% | Many tokens |
| Stripe Crypto | Crypto | 1% | Integration |

---

## LemonSqueezy (Recommended for SaaS)

**Why:**
- EU-friendly (handles VAT)
- Built for SaaS
- Simple API
- 5% fee (includes VAT)

**Alternatives:**
- Stripe (2.9% but needs company)
- Gumroad (10% but instant)
